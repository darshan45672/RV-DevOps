## 💰 GitHub Actions Free Tier Limits

GitHub provides **generous free minutes** for public and private repositories:

| Account Type | Free Minutes/Month | Notes |
|--------------|-------------------|-------|
| **Public repos** | Unlimited | ✅ Always free |
| **Private repos (Free)** | 2,000 minutes | Per account |
| **Private repos (Pro)** | 3,000 minutes | Per account |

**Tips for free tier:**
- Use caching to speed up builds
- Optimize workflows to reduce build time
- Use matrix builds efficiently
- Consider self-hosted runners for unlimited usage
